Demande de suppression de compte et de données

Pour Astral Solitaire, les données liées à votre compte concernent la sauvegarde cloud (progression, statistiques) si vous vous êtes connecté avec Google.

Pour demander la suppression de ces données :

  • Envoyez un e-mail à astralsolitaire.support@gmail.com avec l’objet « Suppression de compte Astral Solitaire ».
  • Indiquez l’adresse e-mail du compte Google utilisé dans l’application (ou un moyen de vous identifier).
  • Nous supprimerons vos données de sauvegarde sous 30 jours.

Vous pouvez aussi vous déconnecter dans l’application (paramètres → Compte). Les données restent alors stockées jusqu’à votre demande de suppression définitive par e-mail.
